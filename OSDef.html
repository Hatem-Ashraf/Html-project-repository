What is an operating system

When the first generation of computers was designed to perform specific operations, for example, a computer designed to perform complex calculations, but with the successive development of computer technology and the emergence of many physical components that are used in the implementation of procedures required of each of them requires a special program to carry out his mission as is Required, and the need arises to organize the work of these various units in a way that prevents their contradiction and thus fails to perform any task that you are asked to do. Based on this need, the term operating system appeared

The operating system
is the program responsible for controlling the work of the computer and all attached components, whether these components are hardware or software. The operating system is the first program that we deal with when the computer is running and it is also the last program that we deal with when the computer is closed
